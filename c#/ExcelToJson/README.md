1. 去掉天津市
2. 匹配16个区进行分桶，每个区进行对比和查找
3. 去掉区，替换 号楼、门、单元、栋 为 ‘-’，尽量格式化为'2-2-2'这种
4. 对比地址
    a. 对地址进行对比比取分数为score
    b. 去掉街道进行对比取分数为score2
        i. 如果score2 是100，那就是小区和门牌号一样，那么score2就是100，但是如果是公司地址，这里就会有问题，如：河西区湛江路19号 vs 河西区隆昌路19号。所以这里增加判断是否存在 ‘-’，不存在就是公司，去做下一步街道对比
        ii. 取score2 大于95的进行街道对比
            1. 如果其中一个没有街道，100分，如：红桥区大明道梦和园3-705    红桥区 梦和园3号楼705
            2. 如果两个都有，且是包含关系，100分，如：
            3. 剩下的进行百分比计算，比如街道得分是40，那么就乘 0.4
    c. 最终得分 = max(max(score, score2), 最终得分），也就是每次遍历取最高分。
5. 输出报告
